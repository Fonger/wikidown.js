pandoc 中 epub 之 css 設定的注意事項

  • pandoc 的 --epub-stylesheet 實作有問題，指定的 css 不會被用，導致排版不正確
  • 解決方式：在家目錄的 .pandoc 下放入 epub.css 檔案，就會被 pandoc 使用了
  • 參考1：Creating an ebook with pandoc -- http://johnmacfarlane.net/pandoc/epub.html
  • 參考2：1.12.2.1 epub3 --epub-stylesheet ignored -- https://groups.google.com/forum/#!topic/pandoc-discuss/MeB1NY3ibV4
